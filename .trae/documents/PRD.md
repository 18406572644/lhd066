## 1. 产品概述

MockupStudio 是面向独立设计师的样机模板管理与生成全栈系统，提供分类模板上传、在线样机生成、批量导出等核心能力，帮助设计师快速产出高质量效果图。目标用户为独立设计师、小型设计工作室，核心价值在于将繁琐的样机制作流程简化为上传设计图一键生成。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 邮箱注册 | 浏览公开模板、生成样机、管理自己的模板与历史 |
| 设计师 | 邮箱注册 | 普通用户权限 + 上传/管理自己的模板、设置模板权限 |
| 管理员 | 系统分配 | 全部权限 + 用户管理 + 数据统计 |

### 2.2 功能模块

1. **首页/模板库**：模板分类导航、标签筛选、搜索、模板卡片展示
2. **模板上传**：分类选择（海报/手机/电脑/包装）、自定义尺寸、标签添加、权限设置
3. **样机生成器**：上传设计图、选择模板、自动贴合预览、调整位置、导出高清效果图
4. **批量生成**：多模板 + 多设计图组合批量生成
5. **历史记录**：生成记录列表、重新下载、删除管理
6. **模板分享**：生成分享链接、设置有效期、下载模板源文件
7. **数据统计**：模板使用次数、生成次数、热门模板排行

## 3. 核心流程

**样机生成主流程**：用户登录 → 浏览/搜索模板 → 选择模板 → 上传设计图 → Canvas 实时预览贴合效果 → 调整位置/缩放 → 选择导出参数 → 生成高清效果图 → 保存至历史记录 → 下载

```mermaid

    "A[用户登录]" --> "B[浏览模板库]"
    "B" --> "C[搜索/筛选模板]"
    "C" --> "D[选择模板]"
    "D" --> "E[上传设计图]"
    "E" --> "F[Canvas实时预览]"
    "F" --> "G[调整贴合位置]"
    "G" --> "H[选择导出参数]"
    "H" --> "I[生成高清效果图]"
    "I" --> "J[保存至历史记录]"
    "J" --> "K[下载效果图]"
```

**模板上传流程**：设计师登录 → 点击上传模板 → 选择分类 → 配置尺寸 → 上传模板底图 → 定义贴合区域 → 添加标签 → 设置权限 → 提交保存

## 4. 用户界面设计

### 4.1 设计风格

- **风格定位**：极简设计风，模拟专业设计软件布局（类 Figma/PS 灰色工作区）
- **主色调**：纯白 #FFFFFF + 浅灰 #F5F5F7 + 中灰 #E5E5EA + 深黑 #1D1D1F
- **强调色**：#007AFF（关键操作按钮）、#34C759（成功状态）
- **按钮风格**：圆角 6px，扁平化，悬停微阴影
- **字体**：标题使用 SF Pro Display / -apple-system，正文使用 SF Pro Text / -apple-system，中文回退 PingFang SC
- **布局风格**：左侧导航栏 + 顶部工具栏 + 中央画布区 + 右侧属性面板（类 Figma 三栏布局）
- **图标风格**：线性图标，1.5px 描边，统一灰色系

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 首页/模板库 | 分类导航 | 顶部标签切换，选中态底部蓝色线条，12px字重500 |
| 首页/模板库 | 搜索栏 | 圆角搜索框，左侧搜索图标，浅灰背景 |

| 样机生成器 | 画布区 | 中央大画布，浅灰棋盘格背景，拖拽交互 |

| 登录/注册 | 认证表单 | 居中白色卡片，极简输入框，蓝色主按钮 |

### 4.3 响应式

桌面优先设计，最小支持 1280px 宽度。侧边栏可折叠，画布区自适应宽度。平板端隐藏右侧属性面板为抽屉，移动端暂不做深度适配。

### 4.4 动效

- 页面切换：淡入 200ms
- 卡片悬停：上浮 4px + 阴影增强 150ms
- 模态框：从底部滑入 250ms
- 生成预览：Canvas 实时渲染无延迟感
- 进度条：平滑动画过渡
